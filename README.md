# MaxCheckBot
use selenium to check all checkbox
幫你把網頁裡的checkbox（複選框）打勾。

# 需要的套件：
<code>pip install selenium</code>

# 執行方式：
<code>python max_checkbox.py</code>
或 
<code>python3 max_checkbox.py</code>

附註：沒有寫的很好，很多地方應該可以模組化。
